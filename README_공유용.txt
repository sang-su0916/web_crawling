🕷️ 웹 크롤링 자동화 도구 - 비개발자용 가이드
===============================================

이 프로그램은 무엇인가요?
-----------------------

웹사이트의 정보를 자동으로 수집해서 Excel 파일로 저장해주는 프로그램입니다.
프로그래밍 지식이 없어도 쉽게 사용할 수 있게 만들어졌습니다.

어떤 일을 할 수 있나요?
--------------------

📰 뉴스 모니터링
• 네이버뉴스, 다음뉴스 등에서 키워드별 뉴스 자동 수집
• 매일 아침 관심 뉴스를 Excel로 정리해서 받아보기

💰 쇼핑 가격 추적  
• 쿠팡, G마켓 등에서 관심 상품 가격 변동 추적
• 할인 시 자동으로 이메일 알림

📈 주식/코인 모니터링
• 관심 종목의 가격 변동 자동 수집
• 하루 종가, 거래량 등을 Excel로 관리

🏠 부동산 시세 추적
• 관심 지역의 매물 정보 자동 수집  
• 시세 변동 추이를 그래프로 분석

설치 방법 (3분 완료)
-------------------

1. 폴더 압축 해제
2. "완전자동설치.bat" 더블클릭  
3. 화면 지시에 따라 Python 설치 (필요시)
4. 패키지 자동 설치 대기
5. "run_scraper.bat" 실행!

사용 방법
---------

1️⃣ 프로그램 실행 후 "5번" 선택 (설정 파일 생성)

2️⃣ "scraper_config.json" 파일 메모장으로 열기

3️⃣ 원하는 웹사이트 정보 입력:
   - name: 사이트 이름 (아무거나)
   - url: 웹사이트 주소
   - schedule: 수집 주기 ("*/30"은 30분마다)

4️⃣ 다시 프로그램 실행해서 "1번" 선택 (테스트)

5️⃣ Excel 파일 확인 → 데이터 수집 성공!

6️⃣ "2번" 선택하면 24시간 자동 실행

실제 설정 예시
-------------

네이버 뉴스 수집:
{
  "name": "IT뉴스",
  "url": "https://news.naver.com/main/list.nhn?mode=LS2D&sid2=001",
  "schedule": "*/60",
  "enabled": true
}

쿠팡 상품 가격 추적:
{
  "name": "아이폰가격",  
  "url": "https://www.coupang.com/vp/products/아이폰",
  "schedule": "*/120",
  "enabled": true
}
